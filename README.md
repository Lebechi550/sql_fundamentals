## SQL Fundamentals – Practice Scripts

## Overview
This repository documents my progression in SQL for data analysis, starting from core querying techniques and gradually moving into database design and architecture concepts.
The project combines practical querying exercises with schema design, helping build both analytical and structural SQL skills.

### Tools and Environment

* Database Engine: MySQL
* IDE: MySQL Workbench
* Version control and documentation: GitHub 

### Databases Used:
  * Sample Database - sakila
  * Custom Practice database - hng

### Folder Structure & learning progression
The sql_script/ directory follows a sequential naming convention (01_, 02_, etc.) to reflect the natural database development lifecycle.

### Key Learning stages include:
* Schema Definition
Creating tables with appropriate data types such as BIGINT and DECIMAL.
* Relational Integrity
Implementing PRIMARY KEY and FOREIGN KEY constraints to maintain relationships between tables.
* Data Validation
Using constraints like NOT NULL, UNIQUE, CHECK, and DEFAULT to ensure data quality.
* Automation & Reporting
Utilizing AUTO_INCREMENT and creating VIEWS to simplify complex queries and enable dynamic reporting.

### Technical Topics Covered
* Data Manipulation Language (DML): querying, filtering, aggregations, subqueries, and conditional logic
* Data Definition Language (DDL): creating and modifying database structures
* Database Architecture: constraints, relational integrity, temporal data handling, and views for reporting

### Learning Approach

Topic-by-topic practice using real examples, progressing from basic queries to more advanced relational logic. I focus on understanding how SQL solves real business problems, not just the syntax.

My goal is to understand the “why” behind database design decisions—such as using DECIMAL for financial accuracy, CURRENT_TIMESTAMP for transaction auditing and reconciliation, and VIEWS to ensure reporting structures remain up-to-date.

### Who This Is For
This repository is useful for:
* Beginners to intermediate learners looking to build strong SQL fundamentals through practical examples.
* Data analysts practicing real query workflows
* Learners interested in understanding both querying and database structure
